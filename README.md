# SkyFlare
SkyFlare is a cloud service with similar workflow as cloudflare.

## Getting started

This project requires Go to be installed. On OS X with Homebrew you can just run `brew install go`.

To start application do the command :

### RUN

``make dev``

To test application do the command :

### RUN

``make unit-test``

### THEN RUN

``make test`` OR ``make test-verbose`` to run all tests with verbose output
